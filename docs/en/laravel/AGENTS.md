# Instructions for AI Agents - Laravel Guidelines

## Applicability

The documents under this directory (`docs/en/laravel/`) apply **only to Laravel applications**.
If the target project does not use Laravel, ignore the guidelines in this directory.

## Priority

If these guidelines conflict with the general guidelines (directly under `docs/en/`), these Laravel-specific guidelines take precedence.
