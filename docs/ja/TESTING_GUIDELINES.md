# テストコードガイドライン

このドキュメントでは、私たちのプロジェクトにおけるテストコードの作成に関するガイドラインを定めます。

## テストの種類

私たちのプロジェクトでは、主に以下の種類のテストを記述します。

- [ユニットテスト](./UNIT_TEST_GUIDELINES.md)
- [フューチャーテスト](./FUTURE_TEST_GUIDELINES.md)

各テストの詳細なガイドラインについては、それぞれ上記のリンク先ドキュメントを参照してください。

## テストデータ

マスタデータや設定、ユーザデータなど、広くテストで用いるデータは、Mother を作成し再利用する。
